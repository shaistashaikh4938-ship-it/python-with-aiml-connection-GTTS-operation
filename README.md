gTTS Text-to-Speech 🔊



Overview

This project converts text into speech using the Google Text-to-Speech (gTTS) library in Python.

Features

Text-to-speech conversion

MP3 audio generation

Multi-language support

Simple and fast processing


Technologies Used

Python

gTTS


Usage

pip install gtts
python app.py

Purpose

To create a simple application that converts written text into spoken audio.
